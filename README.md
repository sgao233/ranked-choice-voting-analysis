# Ranked Choice Voting Analysis

This project is a Data Science for Social Science analysis focused on public support for ranked choice voting. The project explores how support differs across groups and uses statistical methods to better understand patterns in the data.

## Project Overview

The goal of this project was to analyze support for ranked choice voting and identify which factors may be connected to different levels of support. The analysis includes descriptive statistics, data visualization, and bootstrap confidence intervals.

## Key Questions

- Which groups show higher or lower support for ranked choice voting?
- How does political leaning relate to support for ranked choice voting?
- What patterns appear in the data when comparing support across groups?

## Methods Used

- Data cleaning
- Descriptive statistics
- Data visualization
- Bootstrap confidence intervals
- Group comparison
- Interpretation of results

## Tools Used

- R
- HTML
- Data visualization
- Statistical analysis

## Key Findings

The analysis found that support for ranked choice voting differs across groups. Political leaning appeared to be an important factor, with liberals showing higher support than conservatives in the sample. The project also used bootstrap confidence intervals to better understand uncertainty in the results.
